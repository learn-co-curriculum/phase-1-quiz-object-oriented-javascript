# OOJS: Constructor

How could I find the age of the following Dog instance?

```
class Dog {
  constructor(name, age) {
    this.name = name;
    this.age = age;
  } sayWoof() {
    console.log(`Woof, my name is ${this.name}`);
  }}let spinach = new Dog('Spinach', 4);
```

```
spinach.age
```

## Incorrect

```
spinach.age()
```

```
Dog.age(spinach)
```
