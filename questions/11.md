# OOJS: Static Methods

What is the proper syntax for writing a static method?

## Correct

```javascript
class CommonMath {
  static triple(number) {
    return number * number * number;
  }
}
```

## Incorrect

```javascript
class CommonMath {
  triple(number) {
    return number * number * number;
  }
}
```

## Incorrect

```javascript
class CommonMath {
  triple(number) {
    static return number * number * number;
  }
}
```
